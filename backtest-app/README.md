# Backtesting Application

A React-based backtesting application for algorithmic trading strategies.

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

## Build for Production

```bash
npm build
```

## Features

- Symbol selection with lot size display
- Quick strategy configurations
- Entry and exit conditions builder
- Trade parameters management
- Technical indicators (EMA, SMA, MACD, Bollinger Bands, Parabolic SAR, Super Trend)
- Computation time settings
- Target parameters configuration
- Backtest period selection
- API integration for backtesting submission

## Technology Stack

- React 18
- Tailwind CSS 3
- Modern JavaScript (ES6+)
